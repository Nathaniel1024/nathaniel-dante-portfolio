# Nathaniel Dante Portfolio Website

Responsive personal portfolio website showcasing my software projects, graphic design work, achievements, and contact details.

## Overview

This repository contains my multi-page portfolio site with the following sections:

- **About Me**
- **Projects**
- **My Designs**
- **Contact Me**

The site is built with static HTML pages and Tailwind CSS.

## Features

- Clean and responsive layout
- Profile and social links (GitHub and LinkedIn)
- Project showcase with external live demo links
- Graphic design portfolio section
- Direct contact via email

## Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (used in some linked projects)

## Project Structure

```text
.
├── index.html
├── css/
│   └── tailwind.min.css
├── html/
│   ├── projects.html
│   ├── my-designs.html
│   └── contact-me.html
└── images/
```

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/Nathaniel1024/nathaniel-dante-portfolio.git
   ```
2. Open the project folder.
3. Run the site locally by opening `index.html` in your browser.

## Customization

- Update text and personal info in the HTML files.
- Replace images in the `images/` folder.
- Edit styles through Tailwind utility classes in each page.
